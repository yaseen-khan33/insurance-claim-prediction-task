# 🏥 Insurance Claim Prediction (Machine Learning Project)

## 📌 Overview

This project focuses on predicting **medical insurance charges** based on personal attributes such as age, BMI, smoking status, and region.

A **Linear Regression model** is used to estimate insurance costs. The project also includes data analysis and visualization to understand which factors have the greatest impact on charges.

---

## 🎯 Objective

* Predict insurance claim amounts using machine learning
* Analyze the impact of features like age, BMI, and smoking
* Evaluate model performance using error metrics

---

## 📊 Dataset

* **Name:** Medical Cost Personal Dataset
* **Features:**

  * Age
  * Gender
  * BMI (Body Mass Index)
  * Number of children
  * Smoking status
  * Region
* **Target Variable:**

  * Insurance charges

---

## ⚙️ Technologies Used

* Python 🐍

* Scikit-learn

---

## 🔍 Project Workflow

### 1. Data Loading

* Imported dataset using Pandas

### 2. Data Cleaning & Preparation

* Converted categorical variables into numeric format
* Applied one-hot encoding
* Checked for missing values

### 3. Exploratory Data Analysis (EDA)

* Visualized:

  * Age vs Charges
  * BMI vs Charges
  * Smoking vs Charges
* Generated correlation heatmap

### 4. Model Training

* Used **Linear Regression**
* Split data into training and testing sets

### 5. Model Evaluation

* Evaluated using:

  * MAE (Mean Absolute Error)
  * RMSE (Root Mean Squared Error)

---

## 📈 Key Insights

* 🚬 Smokers have significantly higher insurance charges
* 👴 Age has a strong positive correlation with charges
* ⚖️ Higher BMI leads to higher insurance costs
* 📊 The model provides a reasonable prediction of charges
